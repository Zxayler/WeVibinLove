# WeVibinLove
A Discord Bot which can handle Blind dates and confessions.. More features to come!!


Create a `config.json` file in the ``__database__`` directory.

DATABASE: Mysql 8.0


```json
{
	"endpoint": "endpoint",
	"host": "host",
	"token": "TOKEN HERE",
	"passwd": "DB password",
	"invite": "INVITE TOKEN HERE"
}
```
